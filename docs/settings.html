オプション設定

JSON形式に従って記述してください。JSON形式の詳細は以下のサイトを参考にしてください。
JSONとは？データフォーマット（データ形式）について学ぼう！

year

入学年度です。文字列もしくは整数で記入してください。

例: 2021年度入学の場合
　"year": "2021"
　もしくは、
　"year": 2021

fac

学部です。文字列で、以下の書式(シラバス準拠)に従って記入してください。
大学院 "din"
建築学部 "arc"
デザ工 "dsn"
シス理 "sys"
工学部 "ko1"

例: 工学部の場合
　"fac": "ko1"

clickLoginBtn

ログインボタン自動クリック。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "clickLoginBtn": true

adfsSkip

adfs二段階認証自動スキップ。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "adfsSkip": true

exitSidemenu

左メニューを自動で閉じます。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "exitSidemenu": true

styleSidemenu

左メニューのスタイル変更。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "clickLoginBtn": true

styleExamBtn

テスト・アンケートのスタイルの変更。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "styleExamBtn": true

styleExamImg

テストの画像スタイル変更(実験的)。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "styleExamImg": true

addSubTimetable

メニュー横の時間割表示。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "addSubTimetable": true

changeReportBtn

レポート提出ボタンの変更。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "changeReportBtn": true

styleDialog

おしらせダイアログを大きくする。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "styleDialog": true

hideCompletedReports

完了したレポートをカレンダーに表示しない。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "hideCompletedReports": true

syllBtn

各科目にシラバスボタンの追加。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "syllBtn": true

changeLogout

ログアウトボタンの変更。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "changeLogout": true

setMaxWidth

最大横幅を設定。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "setMaxWidth": true

pageTopBtn

トップへボタンを消す。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "pageTopBtn": true

mouseDown

ホイールクリックで新規タブで開けるようにする。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "mouseDown": true

tasklistDisplay

メニュー横の課題表示。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "tasklistDisplay": true

styleNowPeriod

LMSで現在のコマを目立たせる。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "styleNowPeriod": true

displayName

履修者名表示。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "displayName": true

layoutHome

ホーム画面のスタイル変更。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "layoutHome": true

nickname

ニックネームの設定。文字列型です。

例: "nickname": "芝浦ミドリ"

timesBtnValue

作成時間簡易入力の設定。文字列型です。以下のモードから選択してください。
"mode1" 10分,30分,60分,100分,200分,300分,600分
"mode2" 10分,30分,60分,120分,180分,300分,600分
"mode3" 10分,20分,30分,60分,90分,120分,180分
"mode4" 10分,60分,120分,180分,300分,600分,1000分

例: "timesBtnValue": "mode1"

sliderBarMax

作成時間簡易入力の設定。整数型です。バーの最大値を入力してください。


例: "sliderBarMax": 600

tasklistTranslate

課題一覧の位置調整。整数型です。バーの最大値を入力してください。

例: "tasklistTranslate": 0

fixHeadShadow

ページ上部にあるバナーのスタイル。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "fixHeadShadow": true

deadlinemode

課題一覧の表示方式。文字列型です。
absoluteを含むと、絶対表記されます。relativeを含むと、相対表記されます。どちらも含まれると、両方表示されます。Long,Shortを含ませることで、絶対表示の長さを調整できます。

例: "deadlinemode": "absolute-relative"

notepadMode

ユーザーメモ。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "notepadMode": true

ddSubmission

課題ドラッグ&ドロップ提出。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "ddSubmission": true

headLinkTo

ページヘッダのScombZアイコンのリンク先。文字列型です。リンクを表記してください。

例: "headLinkTo": "/portal/home"

updateClear

更新通知全削除ボタンを追加。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "updateClear": true

maxTaskDisplay

課題一覧の最大表示数。整数型です。

例: "maxTaskDisplay": 16

darkmode

ダークモード。文字列型です。
"relative" OSに合わせる
"normal" ダークモードを適用しない
"dark" ダークモード強制

例: "darkmode": "relative"

dadbugFix

課題削除できないバグの修正。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。

例: "dadbugFix": true

defaultInputName

ファイル名の自動入力設定。文字列型です。

例: "headLinkTo": "AA00000_山田太郎"

adjustTimetableData

時間割に関する設定。boolean型です。オンの場合はtrue、オフの場合はfalseと記入してください。
オブジェクトであることに注意してください。
